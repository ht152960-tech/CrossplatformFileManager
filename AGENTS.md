# 给 AI 编程助手的项目规则

本项目是 Kotlin Multiplatform 文件管理工具。

当前重点是实现本地推荐算法，不要优先做 UI。

推荐算法设计见：

- RECOMMENDATION_ALGORITHM.md

实现要求：

1. 推荐算法必须优先考虑：
    - 文件打开时间间隔规律
    - 文件打开后的后继关系

2. 不要使用服务器。
3. 不要使用深度学习框架。
4. 不要使用复杂第三方库。
5. 第一版全部使用 Kotlin 标准库和内存数据结构。
6. 代码要便于以后替换为数据库存储。
7. 所有算法类应尽量放在 shared/commonMain 中。
